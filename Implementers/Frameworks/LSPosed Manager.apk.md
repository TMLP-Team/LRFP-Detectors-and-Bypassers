Please launch the LSPosed Manager in the form of the parasitic directly via the operation button of the LSPosed module in the system module tab of the root manager or via the LSPosed parasitic shortcut on the launcher desktop. There is no need to install the LSPosed Manager as an application located under the ``/data/app`` directory via the ``.apk`` file in the ``.zip`` package or under the ``/data/adb/lsposed`` directory. 

---

请直接在 root 管理器的系统模块选项卡中通过 LSPosed 模块的操作按钮或在启动器桌面通过 LSPosed 寄生器快捷方式打开寄生器形式的 LSPosed 管理器，无需通过压缩包中或 ``/data/adb/lsposed`` 目录下的 ``.apk`` 文件将 LSPosed 管理器安装为位于 ``/data/app`` 目录下的应用程序。
